# Data Folder

Place the dataset locally in this folder using the following structure:

```text
data/Datasets/
├── PKU/
│   ├── ResActionsImagesM/
│   ├── ResActionsImagesL/
│   ├── ResActionsImagesR/
│   └── PKU_M_CAMERA_TRAIN_TEST_VAL/
├── small_dataset/
│   ├── small_dataset_all_files/
│   └── small_dataset_train_test/
└── pku_actions_id.xlsx
```

The actual dataset files are ignored by Git and should not be uploaded to GitHub.
